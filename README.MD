# Weather Shopper Automation (Python Selenium)

## Project Description

This project contains an end-to-end (E2E) automation test for the
Weather Shopper web application using Python, Selenium, and Pytest.

The test simulates: - Opening the website - Reading temperature -
Selecting products based on temperature - Adding items to cart -
Verifying cart and total - Completing checkout - Validating payment
success

## Prerequisites

- Python 3.8+
- Google Chrome
- ChromeDriver

Install dependencies: pip install selenium pytest

## Run Tests

Run all tests: pytest

Run specific test: pytest tests/test_newe2e.py
